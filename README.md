Recognize sign language gestures in real time with OpenCV by:
- extracting keypoints with MediaPipe Holistic
- using them to build LSTM deep neural network with Tensorflow and Keras
- preprocessing data and evaluating performance with Scikit-Learn and NumPy
- achieving over 85% accuracy
